Политика конфиденциальности для мобильной игры Tower Defense

Дата вступления в силу: 6 апреля 2026 года

Добро пожаловать в наше приложение. Мы уважаем вашу конфиденциальность и стремимся защищать ваши личные данные. Эта политика объясняет, какие данные мы собираем, как их используем и с кем можем делиться ими.

1. Сбор данных

Мы можем собирать следующие данные:

Идентификаторы устройств (например, IDFA, Android Advertising ID)
Данные о взаимодействии с приложением (например, экраны, действия, настройки)
Данные об использовании приложений и рекламы
Информация, предоставленная вами добровольно (например, email при регистрации)

💡 Если ваше приложение использует рекламные сети, такие как AdMob, эти сервисы могут собирать дополнительные данные.

2. Использование данных

Мы используем данные для:

Улучшения работы приложения
Персонализации рекламы
Аналитики и статистики использования приложения
Связи с пользователями (например, уведомления о важных обновлениях)
3. Раскрытие данных третьим лицам

Мы не продаем ваши данные. Мы можем делиться ими:

С сервисами аналитики (например, Firebase Analytics)
С рекламными сетями (например, AdMob)
С партнерами, если это необходимо для предоставления функций приложения
4. Дети

Если ваше приложение предназначено для детей до 13 лет:

Мы не собираем личную информацию о детях без согласия родителей
Реклама в приложении может быть ограничена для детей
Данные собираются только в соответствии с законами о защите детей, например COPPA
5. Безопасность данных

Мы принимаем разумные меры для защиты ваших данных от несанкционированного доступа, изменения или удаления.

6. Права пользователей

Вы имеете право:

Запросить копию ваших данных
Попросить об удалении ваших данных
Отключить персонализированную рекламу через настройки устройства
